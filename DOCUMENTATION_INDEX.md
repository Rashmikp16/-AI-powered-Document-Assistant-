## 🆘 Getting Help

### Before You Search
1. Check [QUICKSTART.md](QUICKSTART.md#-troubleshooting)
2. Check [SETUP.md](SETUP.md#common-issues--solutions)
3. Check [TESTING.md](TESTING.md#troubleshooting)

### If Still Stuck
- Check API logs: `docker-compose logs backend`
- Check browser console: Press F12
- Check requirements: Python 3.11+, Node.js 18+, OpenAI key

- Can't connect to backend → Check [SETUP.md](SETUP.md#common-issues--solutions)
